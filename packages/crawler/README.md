# 🕷️ Crawler Package

A powerful and efficient web crawler service built with FastAPI and GraphQL.

## 🚀 Features

- GraphQL API for flexible data querying
- Domain management and crawling
- Asynchronous processing
- Type-safe implementation

## 🛠️ Tech Stack

- FastAPI
- Strawberry GraphQL
- Python 3.13+
- Docker
